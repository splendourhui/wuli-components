## WULIGlobalLoading

> 全局 loading

### props

- show
  - 说明：显示 & 隐藏
  - 类型：Boolean
  - 必须：是
  - 默认值：
  - 样例：
  ```
    true
  ```

- content
  - 说明：内容
  - 类型：String
  - 必须：否
  - 默认值：''
  - 样例：
  ```
    '正在载入...'
  ```
